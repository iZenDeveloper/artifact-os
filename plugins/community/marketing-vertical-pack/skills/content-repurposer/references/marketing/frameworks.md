# Copy & content frameworks

Use in **Hook lab** and spine. Tag each candidate hook with a framework id.

## One-line spine (always)

```
Core idea → Proof → Insight (drama) → CTA (offer)
```

Map any framework onto this spine — do not abandon spine for a framework acronym.

---

## Core frameworks

### PAS — Problem · Agitate · Solve

| Beat | Job |
|------|-----|
| Problem | Name the pain in the audience’s words |
| Agitate | Cost of living with it (time, money, status, body) |
| Solve | Your offer / insight as the clean exit |

**Best for:** consideration, leads, conversion.  
**Weak:** abstract problem with no agitation.  
**Strong:** problem lands on *you this week*, not “people in general”.

### AIDA — Attention · Interest · Desire · Action

| Beat | Job |
|------|-----|
| Attention | Stop-scroll hook (paradox / cost / almost-loss) |
| Interest | One concrete scene or proof |
| Desire | Outcome they want + portable lesson |
| Action | Keyword + deliverable |

**Best for:** ads, short posts, email subjects→body.  
**Weak:** Desire = feature list. **Strong:** Desire = life after the fix.

### 4Ps (marketing copy) — Promise · Picture · Proof · Push

| Beat | Job |
|------|-----|
| Promise | Specific outcome |
| Picture | Sensory “after” scene |
| Proof | Mechanism, demo, social proof (honest) |
| Push | Single CTA + urgency only if real |

### StoryBrand (lite)

1. Hero = **audience** (not the brand)  
2. Problem (external + internal)  
3. Guide = brand (empathy + authority)  
4. Plan = 2–3 clear steps  
5. CTA = direct + transitional  
6. Success / failure stakes  

**Best for:** landing, LinkedIn authority, brand story carousels.

### BAB — Before · After · Bridge

| Beat | Job |
|------|-----|
| Before | Painful status quo |
| After | Specific better state |
| Bridge | Product / insight / offer |

### FAB — Feature · Advantage · Benefit

Only use when product is known. Always end on **benefit in audience language**, not feature jargon.

### Curiosity gap

Open a loop in line 1; **partial** payoff mid; full insight near end + offer.  
Forbidden: pure clickbait with no payoff (destroys trust / saves).

### Social proof stack

“People like you…” → specific outcome → mechanism → soft CTA.  
No invented logos, % or user counts.

### Paradox / almost-loss (Content Pro native)

See `../repurpose-frameworks.md`. Prefer for awareness + viral science/brand storytelling.

## Framework × funnel (quick pick)

| Funnel | Prefer |
|--------|--------|
| awareness | Paradox, curiosity gap, story, almost-loss |
| consideration | PAS, FAB, proof-heavy AIDA |
| decision | 4Ps, StoryBrand plan + risk reverse, PAS |
| retention | Story, insider tip, soft series CTA |

## Framework × platform

| Platform | Prefer |
|----------|--------|
| XHS | Paradox cover → PAS/story middle → offer last card |
| TikTok | Attention kill (0–3s) → story turn → peak → offer |
| LinkedIn | Paradox AIDA; StoryBrand for thought leadership |
| Threads | Curiosity chain; P1 = strongest framework beat |
| Email | PAS or AIDA; subject = Attention only |
