# Marketing psychology (practical)

Use to **frame** hooks and CTAs — never to deceive.  
Fake scarcity, fake social proof, or invented stats = **ship fail**.

## Six levers (Cialdini, applied)

| Lever | Use when | Honest pattern | Fail pattern |
|-------|----------|----------------|--------------|
| **Reciprocity** | Lead magnet, series, free tip | Give usable value first → soft ask | “Free” that is only a pitch |
| **Commitment** | Micro-yes, checklist, quiz | Small next step consistent with identity | Force multi-step before value |
| **Social proof** | Consideration, ads | Specific peer outcome / “founders like you” | Fake % / celebrity name-drop |
| **Authority** | B2B, LinkedIn, decision | Mechanism, credentials, clear expertise | Jargon wall, unearned “#1” |
| **Liking** | Personal brand, community | Shared stakes, warmth, plain language | Cringe over-familiar + hype |
| **Scarcity / urgency** | Real cohort, real stock, real deadline | State the real constraint | Fake countdown forever |

## Extra levers marketer actually use

| Lever | Honest use |
|-------|------------|
| **Loss aversion** | “What you lose if you wait” — must be true cost |
| **Curiosity** | Open loop with real payoff (see frameworks) |
| **Specificity** | Numbers only if sourced or clearly illustrative *and* labeled |
| **Contrast** | Before/after; weak world vs clear plan |
| **Identity** | “For people who…” — match persona, don’t cosplay |
| **Effort justification** | Small investment (save, reply, DM keyword) before big ask |

## Funnel × psych

| Funnel | Levers to favor |
|--------|-----------------|
| awareness | Curiosity, liking, mild loss (personal bill) |
| consideration | Authority, social proof, contrast |
| decision | Authority, real scarcity, risk reverse, clarity |
| retention | Reciprocity, identity, community liking |

## CTA psych (keyword + offer)

| Weak CTA | Strong CTA |
|----------|------------|
| “Learn more” | “Comment **GUIDE** — nhận checklist 5 bước” |
| “Follow for more” | “Save + follow series *X* (part 2: …)” |
| “Buy now” (cold) | “Book 15’ demo — see [specific outcome]” |

## Personal stakes = psych for Content Pro

Spectacle without **you/body/home** under-triggers loss aversion and memory.  
Always land cost on the reader’s world when objective needs 9+ drama.

## Ethics gate (non-negotiable)

- [ ] No fake social proof  
- [ ] No fake scarcity  
- [ ] No medical / financial guarantees without user-supplied claims  
- [ ] Brand DESIGN.md claims respected (Client brands especially)  

If a lever needs a lie to work → **drop the lever**, rewrite angle.
