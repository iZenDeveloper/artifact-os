# Strategy inputs — Content Pro v2.2

Brand (Design System) = **how it looks/sounds**.  
These fields = **why it ships and who it moves**.

Ask only if missing **and** source alone is not enough to infer. Prefer generate with labeled assumptions over long questionnaires.

## Required strategy block (every pack)

| Field | Values / guidance | Effect on pack |
|-------|-------------------|----------------|
| **objective** | `awareness` · `engagement` · `leads` · `conversion` · `retention` · `authority` | CTA type, proof density, soft vs hard offer |
| **funnel_stage** | `awareness` · `consideration` · `decision` · `retention` | Hook depth, offer strength, jargon level |
| **persona** | One line role + context (e.g. “founder B2B SaaS VN, 0–1 PMF”) | Language, stakes, examples |
| **pain_points** | 1–3 pains (concrete) | Hook angles, middle-card escalation |
| **desire** | One clear want | Portable lesson + offer framing |
| **offer** | Keyword + deliverable (PDF, series, checklist, demo…) | CTA matrix; invent if user only gave a keyword |
| **platform_priority** | Ordered list, default all five | Depth order; deprioritized platforms can be thinner |
| **language** | `vi` · `en` · `zh` · mix | Caption language; keep brand voice |
| **angle_framework** | Optional: `PAS` · `curiosity_gap` · `story` · `social_proof` · `paradox` · `almost_loss` | Primary hook family in Hook lab |

## Objective → CTA defaults

| Objective | Default CTA shape |
|-----------|-------------------|
| awareness | Soft: series / “fact tiếp” / save |
| engagement | Question + reply bait / poll / save checklist |
| leads | Lead magnet / waitlist / DM keyword |
| conversion | Demo / trial / buy / book call (one primary) |
| retention | Community / next lesson / re-use tip |
| authority | Sharp question + soft portfolio / newsletter |

## Funnel stage → content posture

| Stage | Posture | Avoid |
|-------|---------|--------|
| awareness | Story, paradox, personal stakes; low jargon | Hard sell, feature dump |
| consideration | Proof + comparison + trade-off | Vague inspiration only |
| decision | Offer clarity, risk reverse, one CTA | More education without ask |
| retention | Insider tip, status, next step in series | Re-introducing the brand from zero |

## Persona template (one block)

```
Role: …
Context: …
Pain: …
Desire: …
Language/tone: …
```

If unknown: infer from source + brand; label as **Assumed persona**.

## Angle frameworks (hook lab)

| ID | Pattern |
|----|---------|
| paradox | Two truths that clash |
| almost_loss | Near-failure of a vital part |
| PAS | Problem → Agitate → Solve |
| curiosity_gap | Open loop; payoff delayed |
| story | Scene → turn → lesson |
| social_proof | “Others like you already…” + cost of ignoring |

Ship **3 candidates** in Hook lab; primary = strongest for objective + funnel.

## Inference rules (no form spam)

1. Source present → draft strategy block first, show user, then platforms.  
2. Only keyword → invent offer + assumed persona; mark assumptions.  
3. Brand is Personal Minimal → still punch; clean ≠ soft middle.  
4. Client brand → respect claims / tone in DESIGN.md; no fake metrics.
