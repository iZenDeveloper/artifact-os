# Platform specs — Content Pro v2.1

Quality rules: `content-pro-standards.md` (v2.1).  
This file = sizes, lengths, deliverable fields.

## Xiaohongshu (XHS)

| Item | Spec |
|------|------|
| Size | **1080×1440** · **3:4** |
| Cards | **5–7** |
| Cover | Paradox/cost hook + prefer *you* (flat fact = fail) |
| Middle/end | C5 twist + final must punch; no sag vs cover |
| On-image | **≤8–12 words** |
| Caption | Story + bullets + **offer CTA** |
| Tags | **5–8** |
| KPI | Save + comment + share-worthy cover |
| Target score | **≥ 8.5 every card** (calibrated) |

## TikTok

| Item | Spec |
|------|------|
| Ratio | **9:16** |
| Length | **15–25s** (max ~40s) |
| Hook | **0–3s** kill line with cost/paradox/**you** |
| Peak | Energy **10/10** + cinematic peak frame |
| Rhythm | Uneven energy map; change every **3–5s** |
| On-screen | **≤8 words/line** |
| Required | Beat table: spoken · text · **exact visual** · energy 1–10 |
| CTA | Mid + end; keyword **+ offer** |
| KPI | Watch time + share + save |
| Target score | **≥ 8.5** hook and peak (calibrated) |

## LinkedIn

| Item | Spec |
|------|------|
| Length | Prefer **900–1600** chars; max **2200** |
| Hook | Paradox/cost first line |
| Middle | No lecture dump; 2–3 proofs max |
| Close | Sharp open question |
| Carousel | **4–6** · **1080×1080** optional |
| Hashtags | **3–5** |
| Target score | **≥ 8.5** (calibrated) |

## Threads

| Item | Spec |
|------|------|
| Posts | **4–8** |
| Per post | **1–3 sentences** |
| P1 | Strongest hook |
| End | Offer CTA |
| Target score | **≥ 8.0** |

## Email

| Item | Spec |
|------|------|
| Subjects | **3–4** · **4–8 words** (one paradox-led) |
| Body | **120–250 words** |
| CTA | **1 primary offer** |
| Target score | **≥ 8.0** |

## Facebook

| Item | Spec |
|------|------|
| Feed image | Prefer **4:5** · **1080×1350** (or 1:1 **1080×1080**) |
| Caption | Prefer **80–300 words**; first **125 chars** must hook (truncation) |
| Hook | Cost/paradox in line 1; peer tone (not corporate LinkedIn clone) |
| Hashtags | **0–3** (often none); quality > tag spam |
| Reels | **9:16** — may **adapt** TikTok script; platform-native open + CTA |
| CTA | Offer in body + optional first comment |
| KPI | Shares + saves + meaningful comments |
| Target score | **≥ 8.5** (calibrated) |

## YouTube

| Item | Spec |
|------|------|
| Long-form thumb | **16:9** · **1280×720** min (prefer **1920×1080**) |
| Title | **≤70 chars** prefer; paradox/cost + curiosity; no clickbait lie |
| Description | **Hook para** (first 2 lines) + bullets + offer + links |
| Chapters | Optional outline timestamps if script has clear beats |
| Thumbnail text | **≤5 words** on-image; face/focal strong |
| Shorts | **9:16** · **15–60s** — script + visual direction (MVP no render); may adapt TikTok with YT-native title |
| CTA | Subscribe + **one** offer/link |
| Target score | **≥ 8.5** title+thumb+open; Shorts peak same bar as TikTok |

## Export fields (HTML pack)

| Platform | Must export |
|----------|-------------|
| All | Strategy block · Hook lab (weak vs strong) · self-scores · **Ready-to-post** |
| XHS | Cards + caption + tags + offer + alt + best time |
| TikTok | Beat table + B-roll/collage board + caption/hashtags + peak visual + best time |
| LinkedIn | Full post ± carousel + hashtags + best time |
| Threads | Numbered posts (copy-paste) + best time |
| Facebook | Caption + optional image direction (4:5) + hashtags + best time ± Reels note |
| YouTube | Title options + description + thumb direction + (Shorts script if short-form) + best time |
| Email | Subjects + preheader + body + plain-text block |
| Conversion | Keyword → offer matrix |
| Strategy | objective · funnel_stage · persona · pain · offer · platform_priority |
