# Hook scoring — Hook Engine

Calibrate **hostile** (−0.3 optimism).  
Flagship ship intent: **≥ 9.0** on overall hook craft for every **primary** line.

---

## Dimensions (0–10 each)

| Dimension | 10 means | Instant −2 if |
|-----------|----------|----------------|
| **Stop-scroll** | Would pause mid-feed | Soft / generic |
| **Stakes** | Cost lands on you/body/home or clear work identity | Abstract spectacle only |
| **Specificity** | Concrete noun/verb/scene | Vague “better/growth” |
| **Open loop** | Must read next beat | Full answer already |
| **Framework fit** | Angle matches funnel + surface | Random drama vs offer |
| **Surface fit** | Length + job correct | Essay on cover / soft TikTok open |
| **Honesty** | No fake proof / fake urgency | Invented stats |

**Overall hook craft** ≈ weighted mean (favor stop-scroll + stakes + surface fit).

---

## Bands

| Band | Meaning | Action |
|------|---------|--------|
| &lt; 7.0 | Fail | Rewrite from diagnosis |
| 7.0–8.4 | Clean but scroll-past | Add cost + personal bill + loop |
| 8.5–8.9 | Interesting, not flagship | Punch verb; cut soft words; tighten loop |
| **≥ 9.0** | Ship primary | Keep; alts can be 8.7+ |
| 9.5+ | Rare; memorable line | Prefer as cross-surface hero if lengths allow |

---

## Strategy fit score (separate)

| Dimension | Check |
|-----------|--------|
| Objective | Line serves awareness vs leads correctly |
| Funnel | Not hard-closing cold awareness |
| Persona | Language matches assumed audience |
| Offer path | Open loop can land on real offer later |

Marketing pack may score strategy fit even when craft is high — if strategy fit &lt; 8.0, rewrite angle (not just adjectives).

---

## Rewrite protocol (order)

1. Name fail modes (hook-patterns.md)  
2. Force **personal bill** or **almost-loss**  
3. Swap soft adjectives → **cost verbs**  
4. Cut to surface budget  
5. Re-open loop (remove full payoff)  
6. Rescore all dimensions  
7. If still &lt; 9.0: change **framework id**, don’t polish the same line
