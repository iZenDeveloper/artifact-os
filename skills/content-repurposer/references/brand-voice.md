# Brand Voice compliance (Marketing Vertical)

**Brand = active Design System** → `design-systems/<slug>/DESIGN.md`.

Visual tokens without voice = **pack fail** for Content Pro flagship work.

## Mandatory read order

1. **Brand Overview** (top of DESIGN.md) — audience, when to pick  
2. **§8 Voice & Tone** in full — especially 8.1–8.5  
3. **§9 Anti-patterns** — visual + voice  
4. Only then apply Color / Type / Layout  

If DESIGN.md §8 is thin (legacy brand), still obey what exists + Content Pro flat-hook ban; prefer Vertical OS presets when possible.

## §8 structure (Marketing Vertical standard)

| Subsection | Agent use |
|------------|-----------|
| **8.1 Brand Voice** | Personality, 3 keywords, ban list |
| **8.2 Tone by Context** | Pick row matching current beat (hook / problem / proof / CTA / edu / BTS) |
| **8.3 Writing Principles** | Sentence length, address form, emoji, language |
| **8.4 Do’s / Don’ts** | Hard filters before ship |
| **8.5 Example Sentences** | Calibrate phrasing — do not clone verbatim if off-topic |
| **8.6 Brand × Content Pro** | How drama/ethics apply under this brand |

## Brand × skill defaults

| Brand | Prefer for | Hook energy |
|-------|------------|-------------|
| `personal-minimal` | Long-form, LinkedIn thoughtful, email | Punch allowed; clean ≠ weak |
| `personal-bold` | XHS/TikTok/Threads stop-scroll | Kill hooks; short lines |
| `professional-clean` | Client / B2B / decks | Professional paradox; no meme |

## QA (voice)

- [ ] Active DESIGN.md §8 read this turn  
- [ ] Hook matches brand ban list (no hype spam on Minimal; no meme on Professional Clean)  
- [ ] CTA shape matches §8.2 CTA row + offer rules  
- [ ] Address form (bạn / you / team) matches §8.3  
- [ ] Example sentences used as **calibration**, not copy-paste off-topic  
- [ ] No cross-brand voice leak (Bold slang in client pack, etc.)
