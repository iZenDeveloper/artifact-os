# Platform best practices — VN / SEA defaults

Defaults assume **Vietnam market** when language is `vi` or user is VN-focused.  
Adjust times/hashtags for other markets when language is `en` / `zh`.

Pair with `../platform-specs.md` (sizes) and Content Pro craft rules.

| Topic | Practice |
|-------|----------|
| Job | Save + trust + “I might try this” — not pure brand flex |
| Cover | Stop-scroll paradox/cost; face or strong focal; **3:4** |
| Body cards | Escalate; C5 twist screenshot-worthy alone |

## TikTok / Reels / Shorts

| Topic | Practice |
|-------|----------|

## Cross-platform (VN cultural notes)

1. **Respect + peer tone** beats aggressive US-style hype for many local audiences.  
2. **Personal stakes** (family, money, time, face/status) outperform abstract global spectacle.  
3. **Offer** should feel like gift exchange (reciprocity), not trap.  
4. Avoid sensitive political/religious hooks unless user owns the risk.  
5. Don’t invent local celebrity endorsements or hospital/clinic claims.
