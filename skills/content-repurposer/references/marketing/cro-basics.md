# CRO basics (for content + light landing/email/ads)

Goal: **one clear next step**, not more sections.

## One-offer rule

| Rule | Detail |
|------|--------|
| One primary CTA | One keyword / one button / one link job |
| Secondary only | Soft “save / share / reply” — never compete with primary |
| Offer = exchange | User gives attention/email/DM → gets **named deliverable** |

Empty CTA (“learn more”) = marketing score fail.

## Message hierarchy

1. **Hook** — why stop  
2. **Stakes** — why care now (*you*)  
3. **Proof** — why believe (2–3 max)  
4. **Offer** — what they get  
5. **CTA** — exact action  

If hierarchy is inverted (features first), rewrite.

## Friction checklist

- [ ] Jargon the persona wouldn’t say? cut  
- [ ] More than one ask? cut to one  
- [ ] Trust gap (price, risk)? add honest risk reverse if real  
- [ ] Mobile: first screen must stand alone  

## Channel CRO notes

| Channel | CRO focus |
|---------|-----------|
| XHS last card | Offer + how to claim (comment keyword / link in bio if real) |
| TikTok end | Verbal + on-screen CTA; mid-CTA allowed once |
| LinkedIn | Question drives comments; soft offer in last lines or first comment |
| Email | Single button/link; subject must earn open |
| Ads | Hook match to landing first line; one offer |

## A/B that is worth shipping in a pack

Do **not** ship full second packs by default.

Worth shipping:

- **Hook A vs B** (same body) on top platforms  
- **Subject A vs B** for email  
- **CTA keyword** alternate if leads objective  

Not worth: random full rewrites without hypothesis.

## Score gates (with Content Pro)

| Marketing dimension | Ship gate |
|---------------------|-----------|
| Offer clarity | ≥ 8.5 |
| Funnel fit | ≥ 8.0 |
| Ready-to-post complete | all required fields |

Craft 9+ without offer clarity still **fails marketer value**.
