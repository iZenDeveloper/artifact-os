# Pre-ship QA — Content Repurposer (hard gates)

Run **in order** before claiming ship. Any fail → rewrite, do not ship.

Maps to Content Pro + MVP scope + regression polish freeze.

---

## 0. Scope

- [ ] MVP: text + visual direction + script only  
- [ ] No primary MP4 / “video đã render” claim  
- [ ] TikTok has `visual_direction` table + `peak_frame` description  

## 1. Strategy

- [ ] objective · funnel · persona · pains · offer present  
- [ ] Offer = keyword + deliverable (not empty slogan)  
- [ ] Hooks serve objective/funnel  

## 2. Brand voice

- [ ] DESIGN.md §8 read this turn  
- [ ] Ban list respected  
- [ ] Address form (bạn/you/team) matches brand  
- [ ] No cross-brand leak  

## 3. Hook & story

- [ ] Flat-hook ban  
- [ ] Personal stakes early (you/body/home **or** buyer cost for B2B)  
- [ ] Story turn + insight with teeth  
- [ ] Hook lab ≥ 3 pairs · **parity punch** on personal-brand niche  
- [ ] Exactly one primary hook; used on C1 or TT 0–3s  

## 4. Platforms

- [ ] XHS: C1 strong · C5/C6 no sag · on-image ≤8–12 words  
- [ ] TikTok: energy map uneven · peak 10/10 · peak shot **non-substitutable**  
- [ ] LinkedIn: 900–1600 prefer · no lecture · sharp question  
- [ ] Facebook: first 125 chars hook · peer tone · not LinkedIn clone · 4:5 if visual  
- [ ] YouTube: title+thumb honest · description hook lines · Shorts script if short  
- [ ] Threads: P1 punch · last = offer  
- [ ] Email: 3–4 subjects · one CTA  
- [ ] No caption clones across platforms  

## 5. Ready-to-post

- [ ] Caption/script · hashtags · alt · best time per platform  
- [ ] **Caption length (P1):** after CTA, ≤2 support lines  
- [ ] leads/conversion: Variant B hook on top-2 platforms  

## 6. Polish freeze (blocking)

- [ ] **P1** captions tight  
- [ ] **P2** peak frame named (subject+action+camera+freeze)  
- [ ] **P3** hook lab not “light” on any niche  
- [ ] Scores calibrated hostile (−0.3) · overall intent ≥ 9.0  

## 7. Integrity

- [ ] No lorem / fake metrics / fake social proof  
- [ ] Sensitive claims flagged  

## Ship claim

Only if **all** boxes checked:

> Calibrated craft ≥ 9.0 · ready-to-post complete · polish P1–P3 pass · MVP scope OK  

If running **regression suite**: also fill score card in `regression-suite.md` for the brief id.
