# Repurpose frameworks

## Spine first

1. One sentence core idea  
2. Three proof points  
3. One CTA  
4. Three hook angles  

If the spine is fuzzy, do not generate platform copy yet.

## Hook angles

- **Curiosity gap** — incomplete payoff in the first line  
- **Tension** — common mistake / costly myth  
- **Payoff** — concrete result or checklist promise  
- **Identity** — “for [role] who [struggle]”  
- **Contrarian** — challenge a default belief (keep fair)

## Native rewrite checklist

For each platform ask:

1. What does a native post look like here?  
2. What should the first second / first line do?  
3. What is the minimum viable visual?  
4. What CTA is natural (not spammy)?  

## PAS / AIDA mapping (optional)

- **PAS:** Problem → Agitate → Solve (good for LinkedIn + Email)  
- **AIDA:** Attention → Interest → Desire → Action (good for XHS covers + ads)  
- **Listicle:** numbered steps (good for carousels + Threads)

## Brand mode switch

| Mode | Design System examples | Voice |
|------|------------------------|-------|
| Personal calm | `personal-minimal` | clear, essay-like |
| Personal energy | `personal-bold` | hook-heavy, short lines |
| Client / B2B | `professional-clean` | measured, proof-led |

Never mix personal-meme energy into client packs unless the brief says so.
